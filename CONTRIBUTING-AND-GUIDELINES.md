# Contributing to Bonnyville Hub

We welcome contributions! Please follow these rules:

1. **Code Style:** Use consistent formatting (Prettier, Tailwind classes for styling).  
2. **Pull Requests:** Branch off `main`, make PRs for review.  
3. **Issues:** Bug reports, feature requests, or suggestions should be logged in the Issues tab.  
4. **Commit Messages:** Use clear, short messages like `feat: add post card component`.  
5. **Respect Community Guidelines:** No code that bypasses moderation or privacy controls.

Thank you for helping make Bonnyville Hub (or whatever I end up calling it) great!


# Community Guidelines

**Content Rules**
- No explicit 18+ content
- No illegal activity
- Respectful discussion encouraged
- Spam / trolling not allowed

**Moderation**
- Minimal intervention
- Admin only removes posts violating rules
- Users can report posts
- Transparency is key

**Design & UX**
- Maintain premium look & feel
- Follow dark mode + forest green accent theme
- Clean spacing & typography

**Open Source**
- Follow coding standards
- Submit PRs for review
- Document features & changes
