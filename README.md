# Bonnyville Hub

**Vision:**  
A premium, local-first digital hub for Bonnyville where individuals, businesses, and organizations can post updates, events, classifieds, and videos. Open-source, distance-based, and community-focused.

**MVP Goals:**  
- User accounts (signup/login)  
- Posts (update, event, classified, video/YT embed)  
- Follow system  
- Distance-based feed  
- Comments, likes, saves  
- Dark mode default + light toggle  
- Clean premium UI  

**Tech Stack:**  
- Frontend: Tailwind 
- Backend: TBD
- Deployment: TBD
- Open-source: MIT License

**Next Steps:**  
1. Seed content with 4H, businesses, and local users  
2. Launch private beta  
3. Collect feedback & iterate

## Roadmap

### Phase 0: Prep
- Vision doc, repo, branding, Google Form
- Secure domain
- Early user contacts

### Phase 1: Auth & Setup (Week 1)
- Next.js project setup
- Tailwind + dark mode
- Supabase auth (email/password + optional OAuth)
- Geolocation + manual area input

### Phase 2: Posts & Feed (Week 2)
- Create posts: update, classified, event, video (YT embed)
- Distance-based feed
- Feed toggle: Following / Recent

### Phase 3: Profiles & Follow (Week 3)
- Profile page: avatar, bio, posts, followers, following
- Follow/unfollow functionality
- Post card UI

### Phase 4: Interactions (Week 4)
- Likes, comments, save
- Basic notifications
- Reporting system
- Seed 40–60 posts

### Phase 5: Polish & Private Beta (Week 5)
- Mobile refinement
- Dark/light toggle
- Invite early users
- Collect feedback

### Phase 6+: Iteration
- Org hierarchy, events reminders, trending feed, analytics dashboards
- Multi-town expansion
